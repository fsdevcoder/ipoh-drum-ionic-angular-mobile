Define Inventory Pattern
Name
  • Pattern Name is required.
  • Pattern Name cannot be more than {{inventoryPatternNameMaxLength}} characters.
Description
  • Pattern Description is required.
  • Pattern Description must have at least {{inventoryPatternDescMinLength}} characters.
  • Pattern Description cannot be more than {{inventoryPatternDescMaxLength}} characters.
Cost
  • Pattern Cost is required.
  • Pattern Cost cannot be more than {{inventoryPatternCostMaxLength}} characters.
  • Pattern Cost format must be [123] or [123.00].
Selling Price
  • Pattern Selling Price is required.
  • Pattern Selling Price cannot be more than {{inventoryPatternSellingPriceMaxLength}} characters.
  • Pattern Selling Price format must be [123] or [123.00].
Quantity
  • Pattern Stock Quantity is required.
  • Pattern Stock Quantity cannot be more than {{inventoryPatternStockQuantityMaxLength}} characters.
  • Pattern Stock Quantity must contain digits only.
